# 🎬 Netflix Titles Data Analysis Project

This project involves manual analysis of Netflix content using **Google Sheets** and **Microsoft Excel** (no coding). The goal is to explore Netflix's content trends, distribution, and key contributors such as genres, countries, and directors.

---

## 📌 Objectives

- ✅ Count of Movies vs. TV Shows  
- 📆 Identify years with the most content releases  
- 🌍 Analyze top content-producing countries  
- 🎭 Find the most common genres  
- 🎬 Discover the most featured directors and actors  

---

## 📊 Tools Used

| Tool         | Purpose                     |
|--------------|------------------------------|
| Google Sheets | Pivot tables, formula-based analysis |
| Excel        | Data cleaning and formatting |
| GitHub       | Version control & publishing |

---

## 🧾 Dataset Info

- **Source**: [Kaggle - Netflix Titles Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- **File**: `netflix_titles.csv`
- Contains 8,000+ rows of Netflix movies and TV shows metadata.

---

## 🔍 Key Insights

- 🎥 **TV Shows** slightly outnumber **Movies**
- 🗓️ Most content added around **2020–2021**
- 🇺🇸 USA, India, and UK are top content producers
- 🧑‍🎨 Drama is the most featured genre
- 🎬 Some directors and actors appear multiple times
